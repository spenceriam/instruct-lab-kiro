How Instruct-Lab Works

üéØ What is Instruct-Lab?

Instruct-Lab is a testing platform that helps you optimize your AI system instructions by providing quantitative success metrics. Instead of guessing whether your instructions work well, get real data on their effectiveness.

üîÑ How It Works

  • Step 1: Enter your OpenRouter API key (stored only in your browser session)
  • Step 2: Select an AI model to test with
  • Step 3: Input your system instructions
  • Step 4: Provide a test prompt/task
  • Step 5: Get instant success probability with detailed metrics

üìä Evaluation Metrics

We use a dual-model approach to evaluate your instructions:

  • Coherence: How logically structured and clear the response is
  • Task Completion: Whether the AI fully addresses the prompt
  • Instruction Adherence: How well it follows your system instructions
  • Efficiency: Token usage and response conciseness

üîí Privacy & Security

Your data is never stored on our servers:

  • API keys are encrypted and stored only in browser session storage
  • All data is cleared when you close the browser or reset the session
  • No tracking, no cookies, no server-side storage
  • Direct API calls to OpenRouter - we never see your data
